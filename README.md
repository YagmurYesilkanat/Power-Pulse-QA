
# PowerPulse Manual Testing Project

## Project Overview
PowerPulse is a fitness web application that allows users to track daily exercises and nutrition activities.

## Testing Scope
- Functional Testing
- Positive and Negative Testing
- Responsive Testing
- API Testing
- Bug Reporting

## Tools Used
- Jira
- TestRail
- Postman
- Swagger
- Chrome DevTools
- Google Sheets

## Project Outcomes
- Designed and executed 39 manual test cases
- Reported 17 bugs in Jira
- Performed functional, responsive and API testing
- Identified UI and API-related inconsistencies
